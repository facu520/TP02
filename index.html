¿qué es una URL?

una url es una caddena de caracteres que se utiliza para identificar y localizar un recurso en la web

1.2- ¿qué es una url absoluta?

una url absoluta es una url que incluye todos los elementos necesarios para identificar y localizar un recurso en la webde manera unica y completa

1.3- ¿qué es una url relativa?

una url relativa es una url que se usa para referirse a un recurso dentro de el mismo sitio web o directorio, y que no incluye todos los elementos necesarios para asi identificar y localizar el recurso.

1.4- ¿cuál es la diferencia entre una url relativa y una url absoluta?

Una dirección URL absoluta contiene toda la información necesaria para localizar un recurso. Una dirección URL relativa localiza un recurso mediante una dirección URL absoluta como punto de partida

1.6- ¿qué es un directorio? ¿qué relación hay con una carpeta?

Un directorio es un tipo exclusivo de archivo que sólo contiene la información necesaria para acceder a archivos o a otros directorios y la relacion es que son lo mismo El término correcto en realidad es directorio, pero carpeta se usa porque es la forma que se ha encontrado para representar gráficamente a los directorios

1.7- En la estructura de un proyecto web, ¿a qué se llama directorio raíz? ¿Con qué símbolo se representa?

el directorio raiz es el directorio principal que contiene todos los archivos y subdirectorios del proyecto. es el directorio mas alto en la jerarquia de directorios del proyecto. se representa con el simbolo barra diagonal /
